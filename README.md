# ThreatMail AI — CTI Escalation Investigation Assistant

🚀 Paste an escalation email → get full RCA, threat analysis, and detection gaps in seconds.

An AI-powered tool that analyzes escalation emails and generates structured threat intelligence, Root Cause Analysis (RCA), and detection gap insights.

---

## 🚀 Overview

In Cyber Threat Intelligence (CTI) workflows, escalation emails are often:

* Unstructured
* Multi-asset (domains, apps, social media, messaging, dark web)
* Time-sensitive

This tool automates the investigation process.

➡️ Paste an escalation email
➡️ Get full threat analysis + RCA in seconds

---

## 🔍 Features

* Extracts threat assets (phishing domains, apps, social media, messaging channels)
* Identifies attack type and affected platforms
* Generates full Root Cause Analysis (RCA)
* Explains detection gaps clearly
* Provides corrective and preventive actions
* Tracks escalations via dashboard
* Supports VirusTotal enrichment (optional)

---

## 🧠 Key Insight

Most missed detections are not due to lack of data —
they occur due to fragmented monitoring across platforms.

This tool helps unify that analysis.

---

## ⚙️ Setup

1. Clone the repository:
   git clone https://github.com/sunidhjb/threatmail-ai.git

2. Install dependencies:
   pip install -r requirements.txt

3. Add API keys:

   * Anthropic API Key (required)
   * VirusTotal API Key (optional)

4. Run the app:
   python app.py
