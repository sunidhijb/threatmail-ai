INSTALL & RUN
─────────────
1. Open PowerShell
2. cd Desktop\ThreatMail_AI
3. pip install -r requirements.txt
4. python app.py
5. Open Chrome → http://127.0.0.1:5000


USE THE APP
───────────
1. Enter Anthropic API key (get at console.anthropic.com)
2. Enter VirusTotal API key (optional, free at virustotal.com)
3. Paste escalation email
4. Click Analyze Escalation
5. Download Word or PDF report


TROUBLESHOOTING
───────────────
"python not recognized"      → Reinstall Python, check "Add to PATH"
"No module named flask"      → Run: pip install -r requirements.txt
API Error 401                → Wrong Anthropic API key
API Error 402                → Add credits: console.anthropic.com/billing
"Could not resolve auth"     → API key not entered in the UI field
Nothing happens on click     → Check PowerShell for error messages
